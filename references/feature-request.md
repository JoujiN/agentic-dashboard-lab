# Feature Request: Dashboard Widget Copy Update

## Requirement
Update dashboard widgets to use more user-friendly language.

## Rules
- Headings should remain short and clear
- Avoid technical wording like "Account Summary"
- Prefer plain language (e.g. "Your Plan", "Your Activity")
- Do not change data or behavior, only text content
- Tests should still pass after changes

## Goal
Improve readability without changing structure or logic.
