# Ticket: Dashboard Recommendation Feels Too Harsh

## Background
Some users said the dashboard feels discouraging when they are behind.

## Request
Adjust the recommendation experience so it stays clear, but feels more supportive.

## Constraints
- Keep the dashboard simple
- Do not remove the recommendation
- Do not change authentication
- Do not add backend logic
- Prefer the smallest possible change

## Notes
- Current status visibility is still important
- The recommendation should still encourage action
